⚡ JavaScript Brings Websites to Life!

JavaScript is the programming language that makes websites interactive and dynamic. While HTML creates structure and CSS adds style, JavaScript adds behavior and functionality.

JavaScript lets you create responsive user interfaces, handle user input, manipulate content in real-time, and build amazing interactive experiences!

📦 Variables Store Information

// Creating variables to store different information let userName = "Alex"; let userAge = 25; let isStudent = true; let favoriteColors = ["blue", "green", "purple"]; // Variables can be changed userName = "Alexandra"; userAge = userAge + 1;
Alex becomes Alexandra, age increases to 26

🔢 Operators Perform Calculations

// Mathematical operations let price = 19.99; let tax = price * 0.1; // 10% tax let total = price + tax; // Comparison operations let canBuy = total < 50; // true or false let message = "Total: $" + total.toFixed(2);
Total calculated: $21.99, canBuy: true

🖥️ Console Shows Output

// Displaying information console.log("Welcome to JavaScript!"); console.log("Current user:", userName); console.log("Shopping total:", total); console.log("Items in cart:", 3); // Debug information console.log("Debug: variables loaded successfully");
All messages appear in browser's developer console

🎯 Different Types of Data

// String (text) let message = "Hello World!"; // Number let score = 100; let price = 29.99; // Boolean (true/false) let isActive = true; let isComplete = false; // Array (list) let fruits = ["apple", "banana", "orange"];
JavaScript handles text, numbers, true/false, and lists automatically
